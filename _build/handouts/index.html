Wanted Queries

Intro

This project will use all of the skills you’ve learned so far this week! The application is a mock up of the software that law enforcement would use to submit a person’s information to see if they have outstanding warrants.

Instructions

Setup

Clone and run the Wanted Queries application by using the following commands in your terminal

  • Navigate to the folder you want to store this inside

  • git clone -b version-1.1 https://github.com/DevMountain-QA/wanted-queries.git

  • cd wanted-queries

  • npm i

  • npm start

  • Review the requirements located here

Steps

  1. Create a new Trello board

  2. Create a plan for your testing

  3. Run your tests

  4. Report bugs, linking your tasks together on Trello

  5. You’re also required to create at least 1 test based on a decision table of Enter Wanted

  6. Last, you’ll also need to write up a report on your testing efforts as if you were sending it to your QA team leader

There is no “bug bounty” or number of tests required. However, you should have enough tracked that we can be confident that you have tested each screen for basic functionality.

Feel free to also include bug reports for improvement recommendations/design flaws.

Add your report and a link to your Trello board to a GitHub repo.
